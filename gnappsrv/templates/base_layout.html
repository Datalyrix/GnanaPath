Graph Config

Gngraph can store the uploaded files as static files and store them into a backend database(postgres)

static mode: Storing graph data as static files

database mode: Storing graph data into backend database

To set these mode Goto Graph Config Section and click the mode. Please note: static mode is turned on by default

Database mode

In the database mode, the graph database will be created and uploaded data will be stored as graph data into the backend database. Currently Postgres is supported as backend database.

To setup Postgres database, Go to Graph Config and then click DB Setup and provide databse connecton credentials, and test connection

also make sure you check dbmode in Graph Config section
